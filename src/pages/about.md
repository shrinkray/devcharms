## About DevCharms

DevCharms is a software development blog and portfolio showcasing custom software solutions and development insights. This site is built with Astro and Tailwind CSS, providing a modern, responsive platform for sharing development knowledge and showcasing projects.

The site features:

- **Blog Posts**: Technical articles and development insights
- **Portfolio**: Showcase of software development projects
- **Modern Tech Stack**: Built with Astro, React, and Tailwind CSS
- **Responsive Design**: Optimized for all devices
- **Accessibility**: Built with accessibility best practices in mind
